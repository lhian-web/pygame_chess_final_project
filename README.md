## Project Description

pygameChess is a chess game developed using Python and Pygame. The project allows users to play chess through a graphical interface while implementing game rules and move validation.

## OOP Concepts

### Encapsulation

Chess piece attributes such as color, position, and movement behavior are stored inside class objects.

### Abstraction

Complex move calculations are hidden behind methods that provide a simple interface to the rest of the program.

### Inheritance

A generic Piece class can serve as a parent class for Pawn, Rook, Knight, Bishop, Queen, and King classes.

### Polymorphism

Different chess piece classes can implement the same method, such as get_valid_moves(), while producing different behaviors.

## Educational Purpose

This repository is used for educational analysis of Object-Oriented Programming concepts.
